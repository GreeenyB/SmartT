# SmartT UI

Editable dashboard source for SmartT fuel intelligence interfaces.

- `dashboard/` is the ESP32 embedded dashboard UI source and fallback reference.
- `local-dashboard/` is the SmartT Local Fleet Dashboard source for laptop/server use.

Open the Local Fleet Dashboard directly in a browser:

```text
local-dashboard/index.html
```

Both interfaces use plain HTML, CSS, and JavaScript. The Local Fleet Dashboard uses Leaflet with OpenStreetMap tiles and does not require an API key.
